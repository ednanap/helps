Projects

Project 1: 3D Portfolio Website -Three.js,WebGL,JavaScript, React Three Fiber, Blender

Right now , I am working on a 3d interactive website! I've already built out the 3D model on Blender — a retro computer. Now I am working on attaching this website as an Iframe on top of the model. An Iframe would be giving my 3D object a live, interactive display. So, instead of just being a static model, the screen or surface can show any webpage, video, or interactive content I want, right within my 3D scene. It is a pretty cool way to blend 3D graphics with web development, I hope to be done with this soon!

Project 2: Event Pal Slackbot - Python, Slack, OpenAI API

EventPal is a Slack bot that I built using Python. The idea behind it was to make business interactions a lot more efficient by automating some of the routine tasks that usually take up time. Here’s how it works: EventPal taps into the Slack API to extract information from business cards. Say someone shares a business card in a Slack channel—EventPal can pull out the key details like the person's name, title, and contact information. Then, it takes things a step further by automatically generating a personalized introductory email based on that information.

Project 3:Full-Stack LLM Service on AWS - Python, AWS Lamda, AWS Api Gateway

In this project, I built a full-stack service that uses large language models (LLMs) on AWS. The process involved developing an API to handle user requests, integrating LLMs to generate responses, and deploying the entire service on AWS for scalability. The project required learning and implementing various AWS services like Lambda, API Gateway, and S3, as well as using Python for the backend. The final product is a scalable, cloud-based AI service that can process and respond to user queries using LLMs.
